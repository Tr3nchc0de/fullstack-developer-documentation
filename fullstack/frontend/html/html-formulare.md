# Formulare

Formulare ermöglichen die Eingabe und Übermittlung von Daten durch den Nutzer. Sie sind essenziell für Interaktivität und Datenverarbeitung.

## Beispiele
```html
<form>
  <input type="text" placeholder="Name">
</form>
```
```html
<input type="email" required>
```
```html
<input type="checkbox" aria-label="Newsletter abonnieren">
```
```html
<button type="submit">Absenden</button>
```
