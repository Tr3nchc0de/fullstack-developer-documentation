# Objekt-Methoden

Objekt-Methoden helfen, Eigenschaften und Werte von Objekten effizient zu verarbeiten.

## Beispiele
```javascript
// Object.keys
const keys = Object.keys({ a: 1, b: 2 });

// Object.values
const values = Object.values({ a: 1, b: 2 });

// Object.assign
const ziel = Object.assign({}, { a: 1 }, { b: 2 });

// Objekt Destructuring
const { a, b } = { a: 1, b: 2 };
```
