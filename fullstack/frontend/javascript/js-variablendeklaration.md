# Variablendeklaration

Variablen sind die Grundlage jeder Programmiersprache. In JavaScript gibt es verschiedene Möglichkeiten, Variablen zu deklarieren, die sich in Verhalten und Gültigkeit unterscheiden.

## Beispiele
```javascript
// Mit var (veraltet, funktionsweit gültig)
var name = "Anna";

// Mit let (blockweit gültig)
let age = 25;

// Mit const (blockweit gültig, Wert unveränderlich)
const city = "Berlin";

// Dynamische Typisierung
let value = 42;
value = "Hallo Welt";
```
