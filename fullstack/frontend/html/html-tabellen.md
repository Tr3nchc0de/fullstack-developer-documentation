# Tabellen

Tabellen dienen zur strukturierten Darstellung von Daten und sind in HTML vielseitig einsetzbar.

## Beispiele
```html
<table>
  <tr><th>Name</th><th>Alter</th></tr>
  <tr><td>Anna</td><td>28</td></tr>
</table>
```
```html
<thead>
  <tr><th>Spalte 1</th></tr>
</thead>
```
```html
<tbody>
  <tr><td>Wert</td></tr>
</tbody>
```
```html
<tfoot>
  <tr><td>Summe</td></tr>
</tfoot>
```
