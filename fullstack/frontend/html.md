## HTML5 - Struktur des Web

### Was ist HTML?
HTML ist die Standard-Markup-Sprache zum Erstellen von Webseiten. Sie definiert die Struktur und den Inhalt einer Webseite durch verschiedene Elemente und Tags.

## CSS3 - Das Styling des Web

### Was ist CSS?
CSS ist eine Stylesheet-Sprache, die beschreibt, wie HTML-Elemente auf einer Webseite dargestellt werden sollen.

### Wichtige CSS3-Konzepte
- **Selektoren** - Element-, Klassen-, ID-, Attribut-Selektoren
- **Box Model** - Margin, Border, Padding, Content
- **Flexbox** - Eindimensionale Layouts
- **CSS Grid** - Zweidimensionale Layouts
- **Animationen** - Transitions, Keyframes
- **Responsive Design** - Media Queries, Mobile-First

### CSS-Frameworks
- **Bootstrap** - Komponenten-basiertes Framework
- **Tailwind CSS** - Utility-First CSS Framework
- **Bulma** - Modernes CSS Framework

## Best Practices

### HTML
- Semantische Markup verwenden
- Accessibility beachten (ARIA, Alt-Texte)
- Valides HTML schreiben
- SEO-optimierte Struktur

### CSS
- Mobile-First Ansatz
- CSS-Variablen für Konsistenz
- Modular aufbauen (BEM, SMACSS)
- Performance optimieren
