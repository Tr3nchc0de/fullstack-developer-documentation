# Funktionen

Funktionen sind zentrale Bausteine für wiederverwendbaren Code. Sie ermöglichen Struktur, Modularität und Flexibilität.

## Beispiele
```javascript
// Standardfunktion
function add(a, b) {
  return a + b;
}

// Arrow Function
const multiply = (a, b) => a * b;

// Callback-Funktion
setTimeout(() => console.log("Hallo!"), 1000);

// Funktion als Wert
const greet = function(name) {
  return `Hallo, ${name}!`;
};
```
