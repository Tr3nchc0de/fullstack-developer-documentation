# Semantische Elemente

Semantische HTML5-Elemente geben der Struktur einer Webseite Bedeutung und verbessern die Zugänglichkeit sowie SEO.

## Beispiele
```html
<header>
  <h1>Website Titel</h1>
</header>
```
```html
<nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">Kontakt</a></li>
  </ul>
</nav>
```
```html
<main>
  <article>
    <h2>Artikelüberschrift</h2>
    <p>Inhalt des Artikels...</p>
  </article>
</main>
```
```html
<footer>
  <p>&copy; 2025 Meine Website</p>
</footer>
```
